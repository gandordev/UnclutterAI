# UnclutterAI

Lightweight browser extension that blocks subscription reminders from ChatGPT, Perplexity and Gemini.


# Add your custom rules

To add your own rules to hide **HTML tags** simply access to `main.js` and add to `ELEMENTS_TO_HIDE` whatever tags you want to hide, filtering by class, ids or even attributes.

**You must use DevTools to know these information.**


# How to use

- Locally
    - Clone the project
    - Go to your browser extensions
    - Enable `Developer Mode`
    - Click on `Load decompressed`
    - Select the cloned repo
